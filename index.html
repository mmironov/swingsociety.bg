In this class we explore steps, routines and rhythms, typical for the Vernacular Jazz Dances in the 1920 - 1950 period. In this class we mostly dance individually.

In this class we dive in the details of Lindy Hop, improve on the basics, work on technique and jazz vocabulary and start to explore different rhythmical variations.
